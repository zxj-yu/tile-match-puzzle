# 方块消除大作战 (Tile Match Puzzle)

[English](README.md) | **中文**

一款使用 **Godot 4 / GDScript** 从零开发的层叠式三消解谜游戏。点击层层堆叠的方块，凑齐三个相同即可消除，在限时内清空棋盘。

> 从零开始学习 Godot 引擎，通过持续迭代完成的完整游戏项目 —— 涵盖玩法设计、数值平衡、UI/UX 与存档系统。

## ✨ 核心特性

### 玩法
- **层叠消除机制**：方块多层堆叠、错位遮挡，只能点击未被覆盖的方块，需要规划挖掘顺序
- **收集槽三消**：拿取的方块进入收集槽，凑齐 3 个相同自动消除；槽满即失败
- **双游戏模式**：
  - 🗺 **闯关模式**：18 个手工设计关卡，含爱心、菱形、十字等主题形状布局
  - ♾ **无尽模式**：程序化生成关卡，难度随波次递增，挑战最高纪录

### 系统
- ⏱ **限时挑战**：每关独立倒计时，最后 10 秒红色警告
- ⭐ **星级评价**：按通关用时评定 1-3 星，可重玩刷星
- 🏆 **段位系统**：消除得分跨局累积，从青铜到王者 7 个段位
- 🔥 **连击机制**：快速连续消除触发连击倍率（最高 ×3）
- 🎒 **道具系统**：每关 3 次机会 —— 扩充收纳格 / +5 秒 / 打乱重排
- 💾 **本地存档**：关卡进度、星级、段位、无尽纪录全部持久化
- ⏸ **暂停菜单**：随时暂停、重试或返回主页

### 视觉与反馈
- 粒子爆散 + 飘分动画的消除特效
- 连击时的强化视觉反馈
- 方块生成/合并的弹性动画（Tween）
- 首页 → 关卡选择 → 游戏的完整界面流

## 🛠 技术要点

| 方面 | 实现 |
|------|------|
| 引擎 | Godot 4.x / GDScript |
| 关卡系统 | **JSON 数据驱动**：布局、限时、星级标准全部外置配置，新增关卡零代码改动 |
| 架构 | 信号（Signal）解耦：游戏逻辑、UI、存档各自独立，通过信号通信 |
| 存档 | 全局单例（Autoload）+ JSON 序列化到 `user://` |
| 程序化生成 | 无尽模式按波次动态生成布局，自动保证可解性（方块数为 3 的倍数） |
| 遮挡计算 | 基于层级 + 位置重叠的实时遮挡检测 |
| 特效 | 纯代码动态生成粒子与飘字，静态工具类封装 |

## 🎮 如何运行

1. 下载并安装 [Godot 4.x](https://godotengine.org/download)
2. 克隆本仓库：
   ```bash
   git clone https://github.com/zxj-yu/tile-match-puzzle.git
   ```
3. 用 Godot 打开项目文件夹（选择 `project.godot`）
4. 按 `F5` 运行

## 📁 项目结构

```
├── levels.json          # 关卡配置（布局/限时/星级标准）
├── scenes/              # 场景文件
│   ├── Main.tscn        # 主场景（游戏+UI+菜单）
│   └── Item.tscn        # 方块预制体
├── scripts/
│   ├── GridManager.gd   # 核心玩法：布局生成/遮挡/消除/道具
│   ├── UI.gd            # 界面流转与状态显示
│   ├── SaveManager.gd   # 存档单例
│   ├── LevelSelect.gd   # 关卡选择界面
│   ├── TitleScreen.gd   # 首页
│   ├── Item.gd          # 方块行为
│   ├── FloatingText.gd  # 飘分特效
│   └── BurstParticle.gd # 粒子特效
└── resources/           # 美术素材（Kenney 免费素材）
```
